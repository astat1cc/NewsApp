
# News App

News App that has Breaking News section and allows user to save some News and make search for news.


## Technologies used

- MVVM
- Room
- Retrofit, OkHttp
- Coroutines
- LiveData
- Navigation Component
- Single Activity


## Api used

I used api from https://newsapi.org/
